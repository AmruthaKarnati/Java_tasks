# Task 1: Student Grade Tracker

A Java-based application developed during the CodeAlpha internship to input, manage, and analyze student grades efficiently. This project focuses on data collection, basic statistical computation, and clean data presentation.

## 📌 Features
* **Grade Management:** Input and manage grades for a single student or a batch of students.
* **Smart Analytics:** Automatically calculates and computes the class **average score**, **highest score**, and **lowest score**.
* **Data Storage:** Utilizes Java `arrays` or `ArrayLists` for dynamic and optimized data handling.
* **Summary Reporting:** Generates and displays a clean, readable summary report of all student records.
* **Flexible Interface:** Built with a user-friendly console-based (or GUI-based) interface for seamless interaction.

## 🛠️ Technical Implementation
* **Language:** Java 
* **Data Structures:** `ArrayList` / `Array`
* **Concepts:** Control flow statements, scanner class for input handling, and iterative loops for statistical calculation.
