# Task 2: Stock Trading Platform

An interactive Java application that simulates a real-world stock trading environment. This project focuses on building robust systems using Object-Oriented Programming (OOP) concepts to manage market data, execute trades, and track financial portfolios.

## 📌 Features
* **Market Simulation:** Simulates a basic stock trading ecosystem with dynamic or mock market price fluctuations.
* **Live Market Display:** Features a dedicated market data display to view active stocks and their current prices.
* **Trading Mechanics:** Supports seamless `buy` and `sell` operations for available stocks.
* **Portfolio Tracking:** Keeps a live record of user balance, active stock holdings, and overall portfolio performance over time.
* **Data Persistence (Optional):** Employs File I/O or a local database to save portfolio data across sessions.

## 🛠️ Technical Implementation
* **Language:** Java
* **Design Architecture:** Strictly built using **Object-Oriented Programming (OOP)** principles (Classes for `Stock`, `User`, `Portfolio`, and `Transaction`).
* **Concepts:** Encapsulation, abstraction, collection frameworks, and optional File I/O for state persistence.
